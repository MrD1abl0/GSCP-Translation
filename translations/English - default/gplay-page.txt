App for easy management game servers


Supported features:
- Servers monitoring
- Detailed server info
- Resources used by server
- Player Details
- Banning, kicking and muting players
- Chatting with players
- RCON with autocomplete and shortcuts
- Addons manager
- Server map change
- Start/stop server
- Version check
- View and change server cvars
- Home screen widgets
